Hmrc.BDApp.O365 Documentation

This project, and its resulting NuGet package, was developed to enable the remediation of existing BDApps to be able to implement interaction with O365 by providing generic MS Graph API to the developer.

It is required/expected that the developer is familiar with the Microsoft Graph API and how it relates to Azure and O365.

The NuGet package does not provide for specific application scenarios. Those scenarios need to be part of the application or as an extension to this package.
